# 🚓 Code Highway Patrol

> Make your agents obey the law of your codebase.

CHP rides shotgun on every agent turn, catching half-baked diffs before they hit `main`. Cleaner output, faster reviews, fewer 3am rollbacks.

## 🧭 What it does

- 🚦 **Stops bad turns at the line.** Type errors, lint violations, failed tests get blocked before the agent commits.
- 🛠️ **Auto-patches the easy stuff.** Narrows `any`, adds null guards, strips dead imports. The agent sees the fix, not a lecture.
- 📋 **Cites the rule that fired.** Real diagnostics, real line numbers. No "the model thought this was wrong."
- 🪝 **Drop-in hook.** Works with Cursor, Windsurf, Codex, and Claude Code. Zero config to start.
